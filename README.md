# TQQQvsQQQ

A small analysis project comparing leveraged ETF TQQQ against QQQ and simple allocation strategies. Includes simulation code, results, and supporting CSVs/plots.

## Requirements

- Python 3.8+
- Install dependencies:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1  # PowerShell
pip install -r requirements.txt
```

## Quick start

Run the main simulation script (outputs CSVs and plots):

```powershell
python sim.py
```

Adjust parameters inside `sim.py` or the source to change horizons, allocations, or plotting options.

## Files

- `sim.py` — main simulation script
- `requirements.txt` — Python dependencies
- `allocation_returns.csv` — per-allocation returns (generated/used)
- `simulation_results.csv` — aggregated simulation outputs
- `strategy_metrics.csv` — calculated performance metrics
- `*.png` — plots generated by the analysis
